#  HỆ THỐNG QUẢN LÝ ĐẠI LÝ Ô TÔ  
**Car Showroom Management System**

Ứng dụng Web hỗ trợ quản lý xe, khách hàng, đơn đặt hàng và nâng cao trải nghiệm người dùng.  
Dự án được thực hiện trong kỳ thực tập tại **VTI Education (VTI Group)** — theo mô hình doanh nghiệp thực tế.

---

##  Giới thiệu dự án
- Thuộc chương trình: **Phát triển ứng dụng thực tế cho doanh nghiệp**
- Xây dựng hệ thống web từ frontend → backend → deploy
- Áp dụng quy trình làm việc **Agile + Scrum**
- Làm việc nhóm, sử dụng GitHub, Jira, Cloud Server, Nginx
---

##  Công nghệ sử dụng

###  Frontend
- ReactJS
- React Router
- Hooks (useState, useEffect)
- Giao tiếp REST API
- SPA (Single Page Application)

###  Backend
- Spring Boot (Java)
- Spring MVC
- JPA / Hibernate
- RESTful API
- PostgreSQL connection

> Trong quá trình nghiên cứu, nhóm cũng thử nghiệm Node.js + Express để so sánh công nghệ.

###  Database
- PostgreSQL
- CRUD
- Quan hệ bảng
- SQL Query

###  DevOps & Triển khai
- Ubuntu Server (Cloud)
- Nginx Reverse Proxy
- Cloudflare DNS
- Domain riêng
- SSL HTTPS
- Terminus
- Git / GitHub

###  Công cụ khác
- Jira (Scrum, backlog, user story)
- VS Code & IntelliJ IDEA
- pgAdmin4
- Google Docs / Google Sheet
- Zalo, Google Meet trao đổi
